## 概要
Microsoft Teamsでは無操作状態が約5分間続くと、ユーザーステータスが自動的に退席中に移行します。  
本プログラムでは2分毎にマウスカーソル位置をチェックし、2分間マウスカーソルの移動がなければ(無操作状態)自動でマウスカーソルを右下に少し移動させることで離席状態になるのを防ぎます。  

## 動作環境
- Windows 10 Home(19042.685)
